# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app


### Users
- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Feed
- `GET /api/user/#/feed`

### Post
- `POST /api/post`
- `PATCH /api/post/#`

### User Blog posts
- `GET /api/user/#/posts`

### Likes
- `GET /api/user/#/likes`

### Follows and followers
- `GET /api/user/#/followed`
- `GET /api/user/#/followers`

**Bonus**
### Tags
- `GET /api/tags`

### Explore
- `GET api/posts/index`
